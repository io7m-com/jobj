com.io7m.jobj

The jobj package implements a parser for the useful subset of the Wavefront OBJ mesh format.

Features

  • Hand-written event-based recovering parser: Efficiently parse, accumulating errors along the way, without being tied to any particular AST types.
  • Well tested (99% automated test coverage).
  • OSGi-ready.
  • JPMS-ready
  • ISC license.
